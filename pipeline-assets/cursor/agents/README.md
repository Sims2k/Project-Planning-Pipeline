# Cursor sub-agents

These agent definitions are used by Cursor as **sub-agents** (custom roles). They appear when you open this project in Cursor so you and your team can invoke role-specific help (e.g. product manager, backend developer, UX researcher).

## Contents

- **Product & strategy**: product-manager, project-manager, business-analyst, market-researcher
- **Design & research**: ux-designer, ux-researcher, ui-developer
- **Engineering**: backend-developer, frontend-developer, solutions-architect, code-reviewer, qa-engineer
- **Documentation**: technical-writer
- **Process**: scrum-master
